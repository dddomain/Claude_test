# TaskGrid — オフラインタスク管理アプリ

シンプルでミニマルなデスクトップファーストのタスク管理アプリです。
サーバー不要・インストール不要。`index.html` をブラウザで開くだけで動作します。

---

## 起動方法

```bash
# リポジトリをクローン（またはファイルをダウンロード）
git clone <repository-url>
cd Claude_test

# index.html をブラウザで開くだけ
open index.html        # macOS
start index.html       # Windows
xdg-open index.html   # Linux
```

外部ライブラリ・CDN・サーバーは一切不要です。

---

## 画面構成

### ダッシュボード

| 要素 | 内容 |
|---|---|
| KPIカード | 総タスク数・完了・進行中・未着手をひと目で確認 |
| 完了率バー | アニメーション付きプログレスバーとやる気メッセージ |
| 優先度グラフ | 高・中・低の件数を横棒グラフで表示 |
| カテゴリグラフ | カテゴリ別の件数を横棒グラフで表示 |
| 期限一覧 | 未完了タスクを期限順に表示（超過日数も表示） |

### タスク一覧

| 機能 | 操作方法 |
|---|---|
| タスク追加 | 右上の「+ 追加」ボタン、または `Ctrl+N` |
| インライン編集 | タスク名・カテゴリ・メモのセルをクリック |
| 編集モーダル | 行右端の ✎ ボタン |
| 削除 | 行右端の ✕ ボタン |
| ソート | 列ヘッダーをクリック（クリックごとに昇順/降順切替） |
| フィルタ | 検索ボックス・ステータス・優先度で絞り込み |
| 一括操作 | チェックボックスで複数選択 → 一括削除 / 一括完了 |

## データ仕様

タスクは以下の項目を持ちます。

| フィールド | 説明 | 値の例 |
|---|---|---|
| タスク名 | 必須 | 企画書の作成 |
| カテゴリ | 自由入力（補完あり） | 仕事, 個人, 学習 |
| 優先度 | 高 / 中 / 低 | 高 |
| ステータス | 未着手 / 進行中 / 完了 / 保留 | 進行中 |
| 期限 | 日付 | 2025-12-31 |
| メモ | 自由記述 | 補足情報など |

---

## オフライン対応

データはブラウザの `localStorage` に自動保存されます。
ページを閉じても、次回開いたときにデータはそのまま残ります。

- 保存タイミング：タスクの追加・編集・削除のたびに即時保存
- ストレージキー：`taskgrid_v1`
- 初回起動時はサンプルタスクが10件挿入されます

> データはブラウザのローカルストレージにのみ保存されます。
> 外部サーバーへの送信は一切行いません。

## キーボードショートカット

| ショートカット | 動作 |
|---|---|
| `Ctrl + N` / `Cmd + N` | タスク追加モーダルを開く |
| `Esc` | モーダルを閉じる |
| `Enter` | インライン編集を確定 |
